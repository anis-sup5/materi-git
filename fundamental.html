Rangkuman Materi Git Introduction

Perintah - perintah GIT

  1. Git Init

    Digunakan untuk menginisialisasi repository peritahnya= => "git init"

  2. Git Config
    • Konfigurasi email & username
    • Config global untuk konfigurasi ke server
    • Config local untuk konfigurasi ke local
    • perintah "git config --list" untuk melihat semua konfigurasi
  3. Git Status

    Digunakan untuk mengetahui status file. Status file secara umum ada 3 yaitu:

    • Untracked
    • Staging Area
    • Modified
  4. Git add .

    Digunakan untuk memasukan file staging area agar dapat di commit

  5. Git commit

    Digunakan untuk mencatat perubahan yang dilakukan. Perintahnya yaitu git commit -m "message"

  6. Git log

    Digunakan untuk melihat riwayat commit

  7. Git Clone

    Digunakan untuk mengcopy repo dari server

  8. Git Remote

    Digunakan untuk berinteraksi antara Client dan Server

  9. Git Push

    Digunakan untuk mengirimkan commite ke server

  10. Git Pull

    Digunakan untuk menggambil commit dari server jika ada perubahan dari branch lain
