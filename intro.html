Rangkuman Materi Git Introduction

Pengertian Git

Git merupakan version control system untuk mencatat setiap perubahan file sourcode kita.

Manfaat GIT

  1. Dapat menyimpan semua version sourcecode
  2. Berkolaborasi membangun sistem
  3. Berkontribusi dalam proyek open source
  4. Mempermudah tracking perubahan
